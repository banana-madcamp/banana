## 프로젝트 소개

**BANANA Market**은 중고거래 플랫폼에서 자주 발생하는  
허위 게시물, 부적절한 이미지 업로드 문제를 해결하기 위해 **Google ML Kit 이미지 라벨링** 기술을 도입한 앱입니다.

판매자가 상품을 등록할 때,  
**입력한 태그와 업로드한 이미지의 실제 내용이 일치하지 않으면 등록이 제한**됩니다.  
이를 통해 **거래의 신뢰성과 품질을 높이고**, 사용자 경험을 향상시키는 것이 목표입니다.
